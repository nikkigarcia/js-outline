What is DOM

Document Object Model
It is the browsers internal programatic representation of the web page.

DOM Manipulation

HTML and CSS are parts of the DOM and anytime you make a change to these with Javascript, that is considered Manipulating the DOM. You can change the web page without actually changing the HTML.

You can manipulate the DOM by using these methods:
document.getElementById(id)
element.getElementByTagName(tag)
document.createElement(name)

DOM Event Listeners

Event Listeners :An event listener is a procedure in JavaScript that waits for an event to occur. The simple example of an event is a user clicking the mouse or pressing a key on the keyboard.
element.addEventListener(event, function, useCapture);
